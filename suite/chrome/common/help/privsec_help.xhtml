Rastreo de usuarios

Estas opciones le permiten comunicar sus preferencias de rastreo a los sitios web. El rastreo es un término que incluye muchos métodos diferentes usados por los sitios web, anunciantes y otros para saber más sobre los hábitos de navegación de sus usuarios (como usted)). Esto incluye información sobre qué sitios visita, cosas que le gustan o disgustan, y las que compra, afectando así a su privacidad. A menudo usan esta información para mostrar anuncios, productos o servicios específicamente dirigidos a usted. En esta sección están disponibles las siguientes opciones:

  • Anunciar a los sitios web mis preferencias de rastreo: marque esta casilla para indicar a &brandShortName; que informe a cada sitio web que visite (así como a sus anunciantes y otros proveedores de contenidos) si quiere o no que se rastree sus hábitos de navegación:
    • No quiero ser rastreado: seleccione esta opción para enviar una solicitud Do-not-track a cada sitio web visitado.
    • Consiento ser rastreado: seleccione esta opción para indicar explícitamente a cada sitio web visitado que está de acuerdo con que se rastreen sus hábitos de navegación.

Notas:

  • Respetar una solicitud Do-not-track es voluntario, por lo que los sitios web no están obligados a respetarla. Los sitios que sí tienen en cuenta la solicitud Do-not-track deberían dejar de rastrear automáticamente su comportamiento sin ninguna acción adicional por parte de usted.
  • Solicitar Do-not-track de un sitio web no afectará a su capacidad de iniciar sesión en él ni hará que &brandShortName; olvide su información privada (como el contenido de un carrito de la compra, información de ubicación o información de inicio de sesión).
  • Puede ver anuncios menos relevantes (es decir, menos específicos) en los sitios web si tiene la opción Do-not-track seleccionada.

Navegación segura

Algunos sitios web incluyen contenido potencialmente peligroso y pueden haber sido identificados como maliciosos. Las preferencias localizadas en la parte inferior del panel Privacidad y seguridad le permiten seleccionar qué tipos de sitios identificados de esta forma serán bloqueados. Si se activan &brandShortName; compara la dirección de cada sitio web que vaya a visitar contra listas actualizadas periódicamente. Se emite una advertencia previa a la carga real del sitio identificado, dándole de esta manera la oportunidad de cancelar el proceso de carga. En esta sección están disponibles las siguientes opciones:

  • Bloquear los sitios señalados como atacantes: marque esta casilla si quiere que &brandShortName; compruebe activamente si el sitio web que está a punto de visitar ha sido identificado por intentar interferir con las funciones normales del equipo o intentar enviar datos personales concernientes a usted a terceros no autorizados a través de Internet. Este tipo de malware se usa frecuentemente para robar información personal, enviar correo basura (spam) o distribuir más malware.
  • Bloquear los sitios señalados como falsificaciones web: marque esta casilla si quiere que &brandShortName; compruebe activamente si el sitio web que está a punto de visitar ha sido identificado como un intento de engañarle para que proporcione información personal tal como un nombre de usuario y una contraseña (a esto se le conoce normalmente como phishing). Estos sitios web imitan a menudo los sitios web legítimos de organizaciones muy conocidas.

Importante: la característica de navegación segura está activada por defecto. No se recomienda desactivar ninguna de las dos funciones ya que puede dar lugar a visitar sitios no seguros.

Nota: la ausencia de una advertencia no garantiza que un sitio sea confiable. Los sitios web maliciosos pueden ser muy difíciles de detectar; incluso un sitio web que parezca seguro puede estar intentando en secreto atacarle o robar su información personal. A veces el propietario del sitio web o su proveedor no saben siquiera que el sitio ha sido comprometido.
